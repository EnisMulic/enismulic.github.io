MSG - Z

A personal discord utility bot written in python with features such as:
  • Logging new users and assigning them a default role
  • Logging deleted and edited messages
  • Adding users to the database
  • Fetching users data from the database
  • Creating and fetching custom data snippets
  • Creating polls
  • Scraping my colleges news site for notifications
  • Scraping youtube RSS feeds
  • Allowing users to change their roles based on academic achivments
